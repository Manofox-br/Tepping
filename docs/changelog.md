# Versões do Teping
**atual: `beta:0.2`**

---

## ✧ beta:0.1 (dia: 26/05/26)

### 🔖│ Variáveis
- adicionado **`var "name" = value`** -> __Define uma variável local__;
- adicionado **`$var(name)`** -> __Obtem o valor da variável local existente__;
- adicionado **`...$$var(name)`** -> __Quado ha mais de um $ ele mostra o texto bruto, só que sem um dos $__;

### 📋│Condicionais
- adicionado **`if condição: {códico}`** -> __Faz uma verificação, se atendida executa o códico das chaves ({})__;

### 📺│Console
- adicionado **`console.print("text")`** -> __Printa um texto mo console. Já vem formatado por padrão__;

---

## ✧ beta:0.2 (dia: 27/05/26)

### 📜│Comentários
- adicionado **`# comentário`** -> __comentário simples, comenta só uma linha__;
- adicionado **`#= comentário =#`** -> __comentário cmplexo, comenta tudo entre os #= e =#__;

### 📚│Função
- adicionado **`def "name"(values): {código}`** -> __cria uma função, ao ser chamada executa o codigo__;
- adicionado **`func "name"(values)`** -> __inicia uma função e passa os valores___;
- adicionado **`return values`** -> __finaliza o codigo (da função) e retorna valores, se for passsado nada ele retornará nada___;

### 🔁│Repetições
- adicionado **`while condição`** -> __repete o código até a condição ser atendida__
- adicionado **`for "var name" in range(repet number): {código}`** -> __repete o número do range e adicionado 1 a variável__

### 🔖│Variáveis
- adicionado **`var "name" += number`** -> __adiciona um número a uma variavél__
- adicionado **`var "name" -= number`** -> __subtraí um número a uma variavél__

### 🧭│Cálculos
- adicionado **`calc(expression)`** -> __calcula uma expressão__

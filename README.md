# Teping Language

A **Teping** é uma linguagem experimental criada em Ruby, com extensão `.tp`.  
Ela permite declarar variáveis, imprimir valores e executar condições de forma simples.

---

## 📒 Instruções

**`README`** -> ~/README.md

**`models`** -> ~/examples/models.md

*~representa a raiz (pasta da linguagem)

---

## 🚀 Instalação

### Windows
1. Instale [Ruby](https://rubyinstaller.org/).
2. Baixe o projeto Teping.
3. Execute o arquivo `teping.bat` para registrar o comando:
   ```bat
   @echo off
   ruby "C:\Program Files\Teping\bin\teping.rb" %1
